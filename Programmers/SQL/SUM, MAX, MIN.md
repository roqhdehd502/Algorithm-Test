## 1. 최댓값 구하기

```sql
SELECT MAX(DATETIME)
FROM ANIMAL_INS
ORDER BY DATETIME DESC;
```

## 2. 최솟값 구하기

```sql
SELECT MIN(DATETIME)
FROM ANIMAL_INS
ORDER BY DATETIME DESC;
```

## 3. 동물 수 구하기

```sql
SELECT COUNT(ANIMAL_ID)
FROM ANIMAL_INS;
```

## 4. 중복 제거하기

```sql
SELECT COUNT(DISTINCT NAME)
FROM ANIMAL_INS
WHERE NAME IS NOT NULL;
```
