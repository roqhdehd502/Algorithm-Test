## 1. 고양이와 개는 몇 마리 있을까

​```sql
SELECT ANIMAL_TYPE, COUNT(ANIMAL_TYPE) AS count
FROM ANIMAL_INS 
GROUP BY ANIMAL_TYPE
ORDER BY ANIMAL_TYPE;
​```

## 2. 동명 동물 수 찾기

​```sql
SELECT NAME, COUNT(DISTINCT(NAME)) AS COUNT
FROM ANIMAL_INS
GROUP BY NAME
ORDER BY NAME
​```
